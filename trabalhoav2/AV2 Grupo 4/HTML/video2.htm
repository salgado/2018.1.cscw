Compartilhamento de tela - Google Docs


O Google Docs é um processador de textos online poderoso, capaz de facilitar o compartilhamento e a edição de arquivos em colaboração. Ele disponibiliza várias funções para que seus usuários configurem as permissões de uso para cada indivíduo que tem acesso a determinados arquivos. Além disso, com o programa, é possível criar uma rede de edição de documentos em escala global, com qualquer pessoa do mundo.
